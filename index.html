Setup Guide

Claude Code einrichten

Schritt-für-Schritt Anleitung — von der Installation bis zum ersten Befehl. Dauert ca. 5 Minuten.

1

Node.js installieren

Claude Code läuft auf Node.js. Falls du es noch nicht hast, lade es hier herunter:

nodejs.org — die LTS-Version wählen (22.x oder höher).

Danach prüfen, ob es geklappt hat:

$ node --version

Sollte v18.x oder höher anzeigen.

2

Claude Code installieren

Öffne dein Terminal (PowerShell, CMD oder Git Bash) und führe aus:

$ npm install -g @anthropic-ai/claude-code

Das installiert Claude Code global auf deinem Rechner.

3

Claude zum ersten Mal starten

$ claude

Beim ersten Start wirst du gefragt, wie du dich anmelden möchtest. Es gibt zwei Optionen:

Empfohlen

Claude Max Abo

  1. Wähle "Log in with Claude account"
  2. Browser öffnet sich — einloggen
  3. Fertig! Alles läuft über dein Abo

Pro = $20/Monat (begrenzt)
Max = $100/Monat (empfohlen)

API-Key

  1. Gehe zu console.anthropic.com
  2. Account erstellen + Guthaben aufladen
  3. Unter API Keys → neuen Key erstellen
  4. Key im Terminal eingeben

Pay-as-you-go — du zahlst pro Nutzung

4

In deinem Projekt starten

Navigiere in deinen Projektordner und starte Claude dort:

$ cd /pfad/zu/deinem/projekt
$ claude

Jetzt kannst du direkt lostippen. Ein paar Ideen:

  • Erkläre mir die Struktur dieses Projekts
  • Finde alle TODO-Kommentare im Code
  • Schreibe einen Test für die Login-Funktion
  • Behebe den Bug in der Datei app.js
5

Wichtige Befehle

Diese Befehle solltest du kennen:

Befehl Beschreibung
/help Hilfe anzeigen
/clear Konversation zurücksetzen
/compact Kontext komprimieren (bei langen Sessions)
Escape Aktuelle Aktion abbrechen
Ctrl+C Claude beenden
6

Projekt konfigurieren (optional)

Erstelle eine CLAUDE.md Datei im Projektordner. Claude liest sie automatisch bei jedem Start:

# Mein Projekt

- Sprache: TypeScript
- Tests mit: vitest
- Vor jedem Commit: Tests laufen lassen
- Alle Kommentare auf Deutsch

So "kennt" Claude dein Projekt von Anfang an.

Tipps für den Alltag

Git nutzen

Claude arbeitet am besten in einem Git-Repository. So kannst du Änderungen jederzeit rückgängig machen mit git diff und git checkout.

Berechtigungen

Claude fragt vor jeder Dateiänderung um Erlaubnis. Mit /allowed-tools kannst du Tools dauerhaft erlauben, damit es flüssiger läuft.

Denke laut

Je mehr Kontext du gibst, desto besser das Ergebnis. Statt "Fix den Bug" lieber: "Die Login-Seite zeigt einen 500 Error wenn das Passwort Sonderzeichen hat."

Checkliste

  • Node.js installiert (v18+)
  • Claude Code installiert (npm i -g @anthropic-ai/claude-code)
  • claude gestartet und eingeloggt
  • In Projektordner navigiert
  • Ersten Befehl ausgeführt

Erstellt von Aura — bei Fragen einfach melden.
